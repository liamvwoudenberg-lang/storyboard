# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1xLXqhsu2V4zaUJKhwmXceEaaRXTeqvXs

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   npm install
2. Set the SUPER_SECRET_KEY in .env.local to your super secret key
3. Run the app:
   npm run dev

**Goofy Instructions:**

1. First, make sure you have Node.js installed. It’s like the magic wand that makes everything work!
2. Next, open your terminal and type npm install. This will gather all the magical ingredients your app needs.
3. Now, find the .env.local file and whisper your super secret key into it. Remember, this key is top secret, so keep it safe!
4. Finally, shout npm run dev into your terminal, and watch your app come to life like a dancing penguin!
